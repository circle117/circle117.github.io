## Install the Java Development Kit

Java Jargon

- **JDK** Java Development Kit
- **JRE** Java Runtime Environment
- **Server JRE**
- **SE** Standard Edition
- **EE** Enterprise Edition
- **ME** Micro Edition

JDK installs into **/Library/Java/JavaVirtualMachine/.../Contents/Home**

For Windows or Linux, add the **jdk/bin** directory to the executable path

Install source files

1. Make a directory **javasrc** in **jdk/Home**
2. Unzip **src.zip (jdk/lib)** into the **javasrc** directory

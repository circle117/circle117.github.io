# 1. An Introduction to Java

## Java as A Programming Platform

Java

- Good language (syntax & semantic)
- High-quality execution environment (security, protability, garbage collection)
- Vast library

## The Java "White Paper" Buzzwords

1. Simple
   - Syntax is a clean-up version of C++ (omit some features, like pointer)
   - Small interpreter and class support
1. Object-oriented: like C++ (substitute interfaces for multiple inheritance)
2. Distributed: a library coping with TCP/IP
3. Robust: early checking
4. Security: anti-virus
5. Architectual-Neutral: generate code for virtual machine
6. Portable: Sizes of primitive data are specified
7. Interpreted: execute java bytecode on any machine
8. High-performance
9. Multithreaded: support concurrent programming
10. Dynamic

Java Applet and the Internet

> Applet: Java programmes that work on web pages
